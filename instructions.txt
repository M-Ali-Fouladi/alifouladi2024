1. Pull the Docker image from DockerHub:

docker pull alifouladi2024/app.py

2. Run the Docker container:

docker run -p 8080:8080 alifouladi2024/app.py

3. Test endpoints:

  - Simple GET request:

    curl localhost:8080/foo
    Response: bar

  - POST request:

    curl -H "Accept: application/json" -H "Content-Type: application/json" -X POST --data "{\"name\": \"Prabh\"}" localhost:8080/hello
    Response: Hello Prabh!

  - Exit container:
  
    curl localhost:8080/kill
